# SmartMeal – Website Prototype

## Projekta mērķis
Izveidot funkcionālu, responsīvu HTML/CSS tīmekļa vietnes prototipu
ar semantisko struktūru un interaktīviem elementiem.

## Funkcionalitāte
- 5 savstarpēji sasaistītas lapas
- Responsīvs dizains
- Interaktīvas pogas un mobilā navigācija

## Tehnoloģijas
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript

- Fonti: Google Fonts (Poppins)
